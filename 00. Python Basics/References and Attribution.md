# References and Attribution

> 이 문서는 `Python Coding Test` 저장소의 Python Basics 문서가 어떤 자료를 참고했고, public repo에서 어떤 기준으로 작성되는지 명확히 남기기 위한 출처/저작권 정책입니다.

## Primary References

- Python 3.14.6 official documentation: <https://docs.python.org/3/>
- Python documentation by version: <https://www.python.org/doc/versions/>
- What's New in Python 3.14: <https://docs.python.org/3/whatsnew/3.14.html>
- Effective Python official website: <https://effectivepython.com/>

## Effective Python Usage Policy

Python Basics의 장/아이템 구성은 `Effective Python, 3rd Edition`의 공식 공개 목차를 학습 체크리스트로 참고합니다.

다만 이 저장소는 책의 대체재나 요약본이 아닙니다.

- 책의 본문 문장, 문단, 설명 흐름을 복사하지 않습니다.
- 책의 예제 코드를 복사하지 않습니다.
- 책의 그림이나 도표를 복사하지 않습니다.
- 공개된 item 제목은 학습 순서와 체크리스트로만 사용합니다.
- 설명, 예제, 비교, 회고는 직접 작성합니다.
- Python 3.14.6 기준으로 공식 문서를 다시 확인합니다.

## Why Keep Attribution?

폴더 구조는 `Python Basics` 자체를 중심으로 두지만, 일부 목차 설계가 Effective Python의 공식 공개 목차에서 출발했다는 사실은 투명하게 남깁니다. 이는 public repo에서 오해를 줄이고, 원저작물에 대한 존중을 표시하기 위함입니다.
