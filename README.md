# Python Coding Test

Python **3.14.6** 기준으로 Python 언어 이해, 자료구조/알고리즘 이론, 문제 해결 패턴, 실제 문제 풀이를 함께 관리하는 포트폴리오형 저장소입니다.

이 저장소의 목표는 단순히 정답 코드를 모으는 것이 아니라, **왜 그런 풀이와 Python 표현을 선택했는지 설명할 수 있는 개발자 노트**를 만드는 것입니다.

## Principles

- **Correctness first**: 풀이보다 먼저 문제 조건, 불변식, 복잡도 한계를 확인합니다.
- **Python 3.14.6 baseline**: 문법과 표준 라이브러리는 Python 3.14.6 공식 문서를 기준으로 검증합니다.
- **Public-ready notes**: 책/강의/웹 자료 원문을 복사하지 않고, 직접 이해한 설명과 직접 작성한 예제를 남깁니다.
- **Portfolio quality**: 접근 과정, 트레이드오프, 복잡도, 실수하기 쉬운 지점을 함께 기록합니다.

## Writing Policy

- Python Basics는 코딩 테스트 팁보다 **Python 언어 모델 이해**에 초점을 둡니다.
- Data Structures와 Algorithms는 **이론, 불변식, 복잡도, Python 구현 전략**을 중심으로 작성합니다.
- Problem Solving Patterns는 **문제 신호와 도구 선택 과정**을 정리합니다.
- Problems는 **실제 문제별 풀이, 코드, 복잡도, 회고**를 기록합니다.
